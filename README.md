# Enterprise Key Site

Apple, Google ve Tesla'ya taş çıkartan enterprise key platformu.

## Özellikler

- Modern ve responsive tasarım
- Framer Motion animasyonları
- Tailwind CSS styling
- TypeScript desteği
- Next.js 14 ile geliştirilmiş

## Kurulum

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Teknolojiler

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
